# WTSMaster - TBC Edition
**by derFlanders**

WTSMaster is a lightweight World of Warcraft: Classic (TBC) Addon designed to simplify selling items in the Trade Channel. It provides a clean UI to manage your items, set prices, and post formatted messages with a single click.
<p><img src="https://media.forgecdn.net/attachments/description/null/description_6d900cfe-24a2-43c3-ba53-25846b00a16b.png" alt="WTSMaster Single" width="300" height="256"></p>

## Features
* **Single & Dual Mode**: Switch between selling one or two items at once.
* **Easy Item Input**: Supports Drag & Drop from bags or Right-Click (when UI is open).
* **Price Management**: Dedicated input fields for Gold, Silver, and Copper.
* **Quick Channel Switch**: Toggle easily between Trade Channels (1/2).
* **Custom Templates**: Use professional presets or write your own custom trade text.
* **Visual Aid**: Integrated Raid Icons for better visibility in chat.
* **Smart Reset**: Automatically clears fields after posting for a faster workflow.
* **Character-Specific Memory**: Saves the window position individually for every character.

## Installation
1. Download the latest release.
2. Extract the folder into your WoW directory: 
   `World of Warcraft\_classic_\Interface\AddOns\`
3. Ensure the folder is named exactly `WTSMaster`.
4. Restart your game or reload your UI.

## Usage
* Type `/wtsm` in the chat to show or hide the main window.
* **Right-Click** an item in your inventory to add it to a slot.
* Set your price and click **Post** to send it to the selected channel.

## Commands
| Command | Description |
| --- | --- |
| `/wtsm` | Toggles the WTSMaster Frame |
